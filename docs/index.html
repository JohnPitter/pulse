Features

Everything you need to manage agents

Full control over your Claude Code instances with a polished, mobile-first interface.

Agent Management

Create and configure agents with custom models, thinking modes, max turns, and granular tool permissions.

Real-time Chat

Stream agent responses with full Markdown rendering, syntax highlighting, and tool call visualization.

Full TTY Terminal

Bidirectional xterm.js terminal with full PTY support. Run any command directly in your agent's environment.

Live Status

Socket.io powered real-time updates. See agent status, token usage, and cost tracking as it happens.

Security First

JWT authentication, bcrypt password hashing, AES-256-GCM encryption, rate limiting, and httpOnly cookies.

Mobile PWA

Installable Progressive Web App optimized for phone-sized screens. Manage your agents on the go.

Quick Start

Up and running in 60 seconds

Three commands. That's all it takes.

1 Clone the repository 
git clone https://github.com/JohnPitter/pulse.git && cd pulse
2 Run the installer 
bash install.sh
3 Start Pulse 
npm start
Or use Docker 
docker compose up -d
